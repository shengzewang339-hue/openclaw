# 图生视频动态提示词方法论（Motion Prompt Methodology）

[第一性原则]
    图片已见原则
        - AI 能"看到"输入图片，不需要重复描述图片内容
        - 只描述"变化"：动作、运动、情绪转变
        - 重复描述图片内容会导致运动减少或异常

    简洁优先原则
        - 最佳长度：20-80 字符（中文）
        - 聚焦 2-3 个核心元素
        - 信息过载会导致模型困惑

    具体动作原则
        - 用具体的物理动作，避免抽象概念
        - "头发随风飘动" ✓
        - "体现自由的本质" ✗

    运动限制原则
        - 镜头运动：最多 1-2 种
        - 主体运动：最多 1-2 种
        - 过多运动会产生混乱

[提示词结构]
    基本公式
        镜头运动 + 主体动作 + 速度/节奏 + 氛围/风格

    完整公式（复杂场景）
        镜头运动 + 主体动作 + 环境动态 + 速度/节奏 + 氛围/风格 + 技术参数

    示例
        基本：镜头缓缓推近，主体转头
        完整：缓慢推镜，主体转头，头发在风中轻轻飘动，黄金时段光线，电影感

[镜头运动词汇库]
    推拉类
        - 推镜头 / 推近：聚焦、紧张、亲密
        - 拉镜头 / 拉远：揭示、疏离、结束感
        - 滑动变焦 / 眩晕效果：心理冲击

    横纵摇类
        - 左摇 / 右摇：环顾、跟随水平运动
        - 上摇 / 下摇：强调高度、规模、威严
        - 快速横摇：转场、惊讶

    环绕类
        - 环绕主体：立体感、动感
        - 弧形运动：戏剧性、强调
        - 半环绕（180度）：常用于对话、对峙
        - 全环绕（360度）：产品展示、英雄时刻

    跟拍类
        - 跟拍镜头：跟随主体移动
        - 背后跟拍：跟随主体背影
        - 侧面跟拍：平行跟随
        - 前方引导：在主体前方引导

    变焦类
        - 变焦推近：强调、戏剧性
        - 变焦拉远：揭示全貌
        - 缓慢变焦：悬疑、张力

    升降类
        - 上升镜头：史诗感、希望
        - 下降镜头：压迫、沉重
        - 航拍镜头：规模、全景

    手持与稳定
        - 手持镜头：真实感、紧张、纪录片风格
        - 固定机位：稳定、庄重
        - 漂浮镜头：梦幻、超现实

    焦点类
        - 焦点转移：引导注意力
        - 浅景深：突出主体
        - 深焦：环境信息丰富

[主体动作词汇库]
    面部动作
        - 缓慢睁眼
        - 微笑浮现
        - 目光转移
        - 眉毛上扬
        - 自然眨眼
        - 表情从X转为Y

    头部动作
        - 转头
        - 微微歪头
        - 缓慢点头
        - 抬头 / 低头 / 移开视线

    身体动作
        - 呼吸起伏
        - 肩膀起伏
        - 前倾 / 后仰
        - 站起 / 坐下
        - 向前走 / 走向镜头
        - 伸出手

    手部动作
        - 举手
        - 手指动作
        - 握紧
        - 说话时比划

    环境互动
        - 头发随风飘动
        - 衣服飘动
        - 布料鼓起
        - 水花飞溅
        - 落叶飘落
        - 烟雾升起
        - 尘埃飘浮

[速度与强度修饰词]
    缓慢/轻柔
        - 缓慢、轻柔、渐渐、柔和
        - 微妙、轻微、细腻
        - 自然、有机、几乎察觉不到
        - 微动、极小幅度

    中等/自然
        - 平稳、稳定、自然
        - 中等、平衡、有控制
        - 流畅、连续

    快速/强烈
        - 快速、迅速、突然
        - 戏剧性、大胆、有力、强烈
        - 爆发、动感、充满活力
        - 大幅度、宏大、史诗感

[氛围与风格词汇]
    电影风格
        - 电影感、影片质感
        - 大片风格、好莱坞质感
        - 独立电影美学
        - 纪录片风格

    导演风格（参考）
        - 克里斯托弗·诺兰风格：史诗、庄重
        - 大卫·芬奇风格：悬疑、精准
        - 泰伦斯·马力克风格：诗意、自然光
        - 韦斯·安德森风格：对称、色彩饱和

    情绪氛围
        - 戏剧性、紧张、悬疑
        - 浪漫、梦幻、怀旧
        - 神秘、诡异、萦绕不去
        - 欢快、充满活力、振奋
        - 忧郁、沉重、沉思

    技术参数（可选）
        - 24帧：电影感
        - 60帧：流畅、慢动作素材
        - 慢动作
        - 胶片颗粒感
        - 镜头光晕
        - 背景虚化光斑

[禁忌规则]
    绝对禁忌
        - ❌ 否定句（不要移动、不要变焦）
            → AI 不理解否定，可能产生相反效果
            → ✓ 用肯定句：镜头保持静止 / 固定机位

        - ❌ 重复图片内容
            → 会导致运动减少或静止
            → ✓ 只描述动作变化

        - ❌ 抽象概念（体现自由、展现喜悦）
            → AI 无法解释抽象意图
            → ✓ 用具体物理动作

        - ❌ 多重场景变化
            → 模型无法处理复杂转场
            → ✓ 单一场景、单一动作链

    避免事项
        - ⚠️ 超过 2 种镜头运动组合
        - ⚠️ 超过 2 种主体运动组合
        - ⚠️ 过长提示词（超过 100 字符）
        - ⚠️ 违反物理规律的动作
        - ⚠️ 与图片内容矛盾的描述

[提示词模板]
    人物肖像动态化
        [镜头运动], [面部动作], [环境动态], [氛围]
        示例：缓慢推镜，眼睛慢慢睁开，头发在风中轻轻飘动，电影感光线

    产品展示
        [环绕运动], [光影变化], [材质强调], [风格]
        示例：360度环绕产品，表面反光闪烁，质感高级，苹果广告风格

    动作场景
        [跟拍运动], [主体动作], [环境反应], [节奏]
        示例：跟拍镜头跟随主体，自信地向前走，尘埃飞扬，高能量

    情绪转变
        [镜头运动], [表情变化], [光影配合], [风格参考]
        示例：缓慢变焦推近，表情从平静转为微笑，暖光增强，亲密肖像感

    环境氛围
        [镜头运动], [环境动态], [氛围元素], [风格]
        示例：缓慢横摇扫过风景，云层飘动，黄金时段光线，史诗电影感

[从分镜到动态提示词的转化]
    输入
        - Beat Board 九宫格提示词（关键帧）
        - Sequence Board 四宫格提示词（段落展开）

    输出
        - 每个 Beat Anchor 一组 Motion Prompt
        - 每组 5 个：1 关键帧 + 4 四宫格
        - 共 9 组 45 个

    转化步骤
        1. 识别画面核心动作
            - 从分镜的"视角"描述中提取关键动作
            - 确定主体运动和镜头运动

        2. 简化为核心元素
            - 删除外观描述（AI已从图片获取）
            - 保留动作、运动、氛围
            - 控制在 2-3 个核心元素

        3. 选择镜头运动
            - 根据情绪选择镜头运动类型
            - 紧张/聚焦 → push in
            - 揭示/疏离 → pull back
            - 动感/立体 → orbit
            - 跟随/沉浸 → tracking

        4. 添加速度修饰
            - 根据情绪节奏选择速度词
            - 平静 → slowly, gently
            - 紧张 → quickly, suddenly
            - 戏剧 → dramatically

        5. 补充氛围（可选）
            - 添加风格参考或技术参数
            - 保持简洁，不超过 1-2 个

    转化示例
        四宫格描述：
        "中景，教室内，林书白站在窗边，阳光斜射进来，他缓缓转头看向门口"

        Motion Prompt：
        "缓慢横摇，主体转头看向门口，阳光从窗户斜射，电影感"
